Windows Installation

1. Download node.exe (http://nodejs.org/dist/v0.5.5/node.exe)

2. Download iisnode (https://github.com/tjanczuk/iisnode)

3. Move node.exe into C:\node directory

4. Run install.bat from iisnode

5. [x64] Do some IIS Magic ;]

Fix for default IIS settings for x64 architecture: 
cscript C:\inetpub\adminscripts\adsutil.vbs SET
        W3SVC/AppPools/Enable32bitAppOnWin64 0

6. Go to http://localhost/node/helloworld/hello.js ... :)
