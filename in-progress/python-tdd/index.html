Basic access authentication

Prosty proces uwierzytelniania oparty o protokół HTTP.

Dostarcza dane uwierzytelniające w żądaniu HTTP (nagłówki).

Dlaczego nie Basic?
  • Opiera się na nadmiarowym protokole (HTTP).
  • Dane przesyłane są w postaci czystego tekstu.
  • Nie możliwości wylogowania się z takiej sesji.
  • Konta użytkowników nie odzwierciedlają żadnych relacji ani nie są z niczym powiązane.
  • Co z innymi zasobami i usługami sieciowymi?
